A. Value Proposition
ChooseWise is designed for students and everyday users who struggle with decisions that involve multiple tradeoffs, like apartments, classes, jobs, or purchases. The problem is that these decisions are often made in an unstructured way, which creates stress and uncertainty. ChooseWise solves this by combining weighted scoring with an AI-generated recommendation, so users can compare options more clearly and understand the reasoning behind the result. This matters because it makes difficult decisions faster, more confident, and easier to explain.

B. Product Overview
- ChooseWise is a web-based decision support tool.
- It allows a user to name two options, enter decision criteria, assign weights, and score both choices.
- The app calculates weighted totals for both options and displays the result in a clear table.
- It also uses the OpenAI API on the backend to generate a recommendation, key reasoning, risks, and a suggested next step.
- The product was built with Flask, Python, HTML, and CSS.
- This is a code-assisted build with a live, functional prototype.

E. Self-Assessment
Biggest remaining product or technical risk: The app still only supports two options and a fixed number of criteria, so it is not yet flexible enough for more complex real-world decisions.
What I would improve with more time: I would add support for more options, dynamic criteria, saved decision histories, and collaborative comparison features.
